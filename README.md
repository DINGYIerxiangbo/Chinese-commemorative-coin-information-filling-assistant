## 简介<br>brief introduction

### 安全性设计/Security Design

域名白名单机制：仅在授权的银行官方网站运行，防止恶意使用

输入验证系统：对所有用户输入进行严格的安全检查和字符过滤

本地数据存储：所有个人信息仅存储在用户本地，不上传至任何服务器

Domain Whitelist Mechanism: Only runs on authorized official bank
websites to prevent malicious use

Input Validation System: Strict security checks and character filtering
for all user inputs

Local Data Storage: All personal information is stored locally on the
user's device and not uploaded to any server

## 插件使用/Plugin Usage

### 一、插件安装与启动/I. Plugin Installation and Launch

安装步骤：

下载插件文件包并解压缩打开Chrome浏览器，进入扩展程序管理页面（chrome://extensions/）开启右上角的"开发者模式"点击"加载已解压的扩展程序"，选择解压后的文件夹安装成功后，浏览器工具栏将显示插件图标

Installation Steps:

Download the plugin file package and extract it.Open the Chrome browser and go to the Extensions management page (chrome://extensions/).Enable "Developer mode" in the top-right corner.Click "Load unpacked extension" and select the extracted folder.After successful installation, the plugin icon will appear in the browser toolbar.

启动方式：

点击浏览器工具栏中的插件图标即可打开主界面插件界面为弹出式窗口，尺寸为360x500像素，优化显示效果

Launch Method:

Click the plugin icon in the browser toolbar to open the main interface.The plugin interface is a pop-up window with dimensions of 360x500 pixels, optimized for display.

### 二、核心功能使用/II. Core Function Usage

#### 1. 个人信息管理/1. Personal Information Management

添加信息：

点击主界面"已保存的信息"区域的"添加"按钮在弹出的编辑窗口中填写以下信息：

  必填项：姓名、身份证号、手机号
  
  选填项：省市、地市、县区、预约网点、预约数量点击"保存信息"完成添加
  
Add Information:

Click the "Add" button in the "Saved Information" section of the main interface.Fill in the following information in the popup editing window:

  Required: Name, ID number, Phone number
  
  Optional: Province/City, City, County/District, Appointment branch, Appointment quantityClick "Save Information" to complete the addition.
  
编辑信息：

在信息列表中点击编辑按钮（铅笔图标）或通过"检查信息"功能进入详细编辑界面

Edit Information:

Click the edit button (pencil icon) in the information list.Or use the "Check Information" function to enter the detailed editing interface.

删除信息：

点击信息项右侧的删除按钮（垃圾桶图标）确认删除操作

Delete Information:

Click the delete button (trash can icon) on the right side of the information item.Confirm the deletion action.

信息排序：

点击"排序"按钮进入排序模式拖拽信息项调整显示顺序再次点击"完成"保存排序

Sort Information:

Click the "Sort" button to enter sorting mode.Drag and drop information items to adjust the display order.Click "Done" again to save the sorting.

#### 2. 银行预约功能/2. Bank Appointment Function

快速访问：

"银行预约链接"区域显示支持的11家银行点击银行名称直接跳转至对应预约页面支持折叠/展开显示，节省空间

Quick Access:

The "Bank Appointment Links" section displays 11 supported banks.Click the bank name to directly jump to the corresponding appointment page.Supports collapse/expand display to save space.

银行排序：

点击银行列表右上角的"排序"按钮拖拽银行项调整顺序完成排序后点击"完成"

Bank Sorting:

Click the "Sort" button in the topright corner of the bank list.Drag and drop bank items to adjust the order.Click "Done" after completing the sorting.

特殊银行说明：

陕西农信：点击后显示二维码，需手机扫描预约

Special Bank Instructions:

Shaanxi Rural Credit Union: Click to display a QR code, which requires scanning with WeChat for appointment.

#### 3. 智能填充功能/3. Smart Fill Function

使用条件：

必须在支持的银行预约页面使用页面完全加载后才能正常填充

Usage Conditions:

Must be used on supported bank appointment pages.The page must be fully loaded for the fill function to work properly.

操作步骤：

打开目标银行的纪念币预约页面在插件主界面点击信息项右侧的"填入"按钮系统自动识别并填充表单字段查看填充结果提示，确认填充成功

Operation Steps:

Open the commemorative coin appointment page of the target bank.Click the "Fill" button on the right side of the information item in the plugin main interface.The system automatically identifies and fills in the form fields.Check the fill result prompt 
to confirm successful filling.

#### 4. 数据管理功能/4. Data Management Function

导出数据：

点击底部工具栏的"导出"按钮自动生成JSON格式文件下载文件包含所有已保存的信息数据

Export Data:

Click the "Export" button in the bottom toolbar.Automatically generate and download a JSON format file.The file contains all saved information data.

导入数据：

点击"导入"按钮选择备份文件系统验证数据格式并导入有效信息显示导入结果统计

Import Data:

Click the "Import" button to select a backup file.The system verifies the data format and imports valid information.Display import result statistics.

功能开关：

自动获取验证码：开启后尝试自动识别短信验证码自动填充：进入预约页面时自动触发填充功能

Function Switches:

Automatic verification code retrieval: When enabled, attempts to automatically identify SMS verification codes.Automatic fill: Automatically triggers the fill function when entering the appointment page.

### 三、注意事项/III. Precautions

兼容性要求：

仅支持Chrome、Edge浏览器及基于Chromium的浏览器需要在支持的银行域名下使用填充功能确保浏览器版本较新

Compatibility Requirements:

Only supports Chrome, Edge, and other Chromiumbased browsers.The fill function must be used under supported bank domains.Ensure the browser version is up to date.

数据安全：

所有数据本地存储，不上传服务器输入内容经过安全验证，防止XSS攻击建议定期备份重要数据

Data Security:

All data is stored locally and not uploaded to any server.Input content is securityverified to prevent XSS attacks.Regularly back up important data is recommended.

使用限制：

严禁用于商业用途遵守各银行预约规则不得用于违法违规活动

Usage Restrictions:

Strictly prohibited for commercial use.Comply with the appointment rules of each bank.Must not be used for illegal or noncompliant activities.

### 四、故障排除/IV. Troubleshooting

填充失败：

检查当前页面是否在支持的白名单内确认页面完全加载完成尝试手动点击"填入"按钮

Fill Failure:

Check if the current page is within the supported whitelist.Ensure the page is fully loaded.Try manually clicking the "Fill" button.

数据丢失：

检查浏览器存储空间定期使用导出功能备份数据避免同时使用多个清理工具

Data Loss:

Check browser storage space.Regularly use the export function to back up data.Avoid using multiple cleanup tools simultaneously.

界面异常：

重启浏览器重新加载扩展程序检查是否有其他冲突扩展

Interface Abnormalities:

Restart the browser.Reload the extension.Check for other conflicting extensions.
