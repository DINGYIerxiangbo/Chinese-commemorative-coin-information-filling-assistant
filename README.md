纪念币信息填写助手是一款专为纪念币预约设计的Chrome浏览器扩展程序

帮助用户快速、准确地填写银行预约表单

提升预约效率和成功率

The Commemorative Coin Information Filling Assistant is a Chrome browser
extension

specifically designed for commemorative coin reservation

helping users quickly and accurately fill out bank reservation forms to
improve reservation efficiency and success rates

## 简介

### 安全性设计/Security Design

域名白名单机制：仅在授权的银行官方网站运行，防止恶意使用

输入验证系统：对所有用户输入进行严格的安全检查和字符过滤

本地数据存储：所有个人信息仅存储在用户本地，不上传至任何服务器

Domain Whitelist Mechanism: Only runs on authorized official bank
websites to prevent malicious use

Input Validation System: Strict security checks and character filtering
for all user inputs

Local Data Storage: All personal information is stored locally on the
user's device and not uploaded to any server

### 高效便捷/Efficiency & Convenience

一键填充：在支持的银行页面快速填充已保存的个人信息

多信息管理：支持保存多个预约人信息，方便家庭或团体预约

智能匹配：自动识别不同银行的表单结构，提高填充成功率

One-Click Filling: Quickly fill saved personal information on supported
bank pages

Multi-Profile Management: Support for saving multiple reservation
profiles for family or group reservations

Smart Matching: Auto-recognize bank form structures for higher filling
accuracy

### 多银行支持/Multi-Bank Support

目前支持以下银行的纪念币预约页面：

交通银行、中国银行、农业银行、工商银行

建设银行、华夏银行、邮储银行、浦发银行

苏州银行、徽商银行、陕西农信（无网页版）

Currently supports the following bank reservation pages:

Bank of Communications, Bank of China, Agricultural Bank of China, ICBC

China Construction Bank, Huaxia Bank, Postal Savings Bank, SPD Bank

Bank of Suzhou, Huishang Bank, Shaanxi Rural Credit Union (No web
version, WeChat Mini Program only)

### 数据管理/Data Management

导入导出：支持预约信息的备份和恢复

数据验证：自动验证身份证号、手机号等信息的合法性

排序功能：可自定义银行链接和个人信息的显示顺序

Import/Export: Support for backup and restoration of reservation
information

Data Validation: Automatic verification of ID numbers, phone numbers,
other information validity

Sorting Function: Customizable display order for bank links and personal
information
