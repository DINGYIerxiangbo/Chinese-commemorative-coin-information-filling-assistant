纪念币信息填写助手 / Commemorative Coin Information Filling Assistant

简介
纪念币信息填写助手是一款专为纪念币预约设计的Chrome浏览器扩展程序，帮助用户快速、准确地填写银行预约表单，提升预约效率和成功率。

安全性设计
域名白名单机制：仅在授权的银行官方网站运行，防止恶意使用
输入验证系统：对所有用户输入进行严格的安全检查和字符过滤
本地数据存储：所有个人信息仅存储在用户本地，不上传至任何服务器

Project Overview
The Commemorative Coin Information Filling Assistant is a Chrome browser extension specifically designed for commemorative coin reservation, helping users quickly and accurately fill out bank reservation forms to improve reservation efficiency and success rates.

Security Design
Domain Whitelist Mechanism: Only runs on authorized official bank websites to prevent malicious use
Input Validation System: Strict security checks and character filtering for all user inputs
Local Data Storage: All personal information is stored locally on the user's device and not uploaded to any server
